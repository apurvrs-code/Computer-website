Computer Chapter 5 - Forms, Frames, and CSS

Forms Frames CSS

Forms are used to collect user input in HTML.

Common form elements:

  • <input> – text, password, radio, checkbox, etc.
  • <textarea> – multi-line text input
  • <select> – drop-down list
  • <button> or <input type="submit"> – submit button

Example:

<form action="/submit" method="post">
  Name: <input type="text" name="name"><br>
  Email: <input type="email" name="email"><br>
  Gender: 
  <input type="radio" name="gender" value="Male"> Male
  <input type="radio" name="gender" value="Female"> Female<br>
  <input type="submit" value="Submit">
</form>

Frames allow splitting the browser window into multiple sections, each displaying a separate HTML page.

Tags:

  • <iframe> – embeds another HTML page

Example:

<iframe src="page.html" width="400" height="300"></iframe>

CSS is used to style HTML elements (colors, fonts, layout).

Ways to add CSS:

  • Inline – inside element: <h1 style="color:red">Hello</h1>
  • Internal – inside <style> in <head>
  • External – in a separate .css file linked using <link>

Example (Internal CSS):

<style>
  body {
    background-color: #121212;
    color: #ffffff;
  }
  h1 {
    color: #ff00ff;
    text-align: center;
  }
</style>

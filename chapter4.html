Computer Chapter 4 - Images, Links, and Tables

Images Links Tables

The <img> tag is used to display images in HTML.

Attributes:

  • src: path to image
  • alt: description of image
  • width & height: size of image

Example:

<img src="flower.jpg" alt="Flower Image" width="200">

The <a> tag is used to create hyperlinks.

Attributes:

  • href: URL of the link
  • target: where to open link (_blank, _self)

Example:

<a href="https://www.google.com" target="_blank">Visit Google</a>

Tables are used to display data in rows and columns.

Tags:

  • <table>: defines table
  • <tr>: defines table row
  • <th>: defines table header
  • <td>: defines table data cell

Example:

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>15</td>
  </tr>
</table>
